# Multiplayer Tank Game
* Developed a tank game using Java and Java GUI
* Implemented using Object Oriented Principles


# Instructions
* TANK 1

UP - W
DOWN - S
LEFT - A
RIGHT - D
SHOOT - Space

* TANK 2

UP - up
DOWN - down
LEFT - left
RIGHT - right
SHOOT - Enter

* The goal of the game is to destroy the other tank until it does not have any more lives.
